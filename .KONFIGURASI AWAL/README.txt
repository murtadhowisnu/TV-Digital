import database
---------------
1.pada aplikasi phpmyadmin buatlah database baru denga nama "balmon_ukuradio"
2.arahkan ke menu import
3.import database balmon_ukuradio.sql ke phpmyadmin
___________________________________

konfigurasi
---------------------
1.buka file database.php di folder application/config
2.ubah hostname,username,password serta nama database anda
3.setelah itu, buka file config.php di folder yang sama
4.ubah $config['base_url'] menjadi domain yang anda miliki
5.konfigurasi ini berlaku untuk semua aplikasi
